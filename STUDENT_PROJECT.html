üìã Project Scenario

You work for a fashion tech startup developing an AI-powered design assistant. The marketing team needs a demo showing how GANs can generate fashion item designs. You've been given a working DCGAN implementation that generates handwritten digits (MNIST), and your task is to adapt it to generate fashion items (Fashion-MNIST) and create a comparative analysis tool.

üìö Background

The provided codebase includes a fully functional DCGAN implementation with:

  • Backend (FastAPI + PyTorch): DCGAN models, training loop, REST API, WebSocket support
  • Frontend (React + Vite): Interactive dashboard, real-time visualization, device selection
  • Features: GPU/CPU switching, model save/load, live metrics, training logs

MNIST Dataset (Baseline)

  • Handwritten digits (0-9)
  • 28√ó28 grayscale images
  • 60,000 training samples
  • Simple patterns, high contrast
  • Fast convergence (~5-10 epochs)

Fashion-MNIST (Your Task)

  • Clothing items (10 categories)
  • 28√ó28 grayscale images
  • 60,000 training samples
  • Complex textures, varied shapes
  • Slower convergence (~15-25 epochs)

Fashion-MNIST Classes: T-shirt/top, Trouser, Pullover, Dress, Coat, Sandal, Shirt, Sneaker, Bag, Ankle boot

üÜò Troubleshooting Tips

Backend Port Issues: If port 8000 is in use, run lsof -ti:8000 | xargs kill -9
Training Too Slow: Make sure GPU (MPS) is selected, not CPU. Check device indicator shows "‚ö° GPU (MPS)"
WebSocket Disconnects: This is normal during very fast GPU training. Refresh the browser if needed.
Poor Image Quality: Fashion-MNIST needs more epochs. Try 20-30 epochs for better results.
React Component Issues: Follow existing component patterns in App.jsx. Use same styling classes.
